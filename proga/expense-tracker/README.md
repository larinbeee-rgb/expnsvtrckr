Expense Tracker — Менеджер расходов

Автор - Баранов Д.С.

Зачем это?

Записывать каждый потраченный рубль
Не давать себе соврать (валидация не пропустит "абвгд" в поле суммы)
Искать, сколько ты потратил на кофе, дошики и такси
Не потерять данные — они сохраняются в JSON

Быстрый старт

1. Скачай код
```bash
git clone https://github.com/твой_логин/expense-tracker.git
cd expense-tracker
2. Запусти main.py
3. Пользуйся
Добавляй расходы - таблица заполняется
